echo "Instalando pip..."
sudo python2 get-pip.py 2>/dev/null | progressbar

echo "Actualizando setuptools y wheel..."
sudo python2 -m pip install -U setuptools wheel 2>/dev/null | progressbar

echo "Instalando paquetes de python..."
python2 -m pip install -U distorm3 yara pycrypto pillow openpyxl ujson pytz ipython capstone 2>/dev/null | progressbar

echo "Instalando yara de nuevo..."
sudo python2 -m pip install yara 2>/dev/null | progressbar

echo "Creando enlace simbólico para libyara..."
sudo ln -s /usr/local/lib/python2.7/dist-packages/usr/lib/libyara.so /usr/lib/libyara.so 2>/dev/null

echo "Agregando Volatility al PATH..."
export PATH="$PATH:/path/to/volatility"

echo "¡Listo! Todos los paquetes han sido instalados correctamente y Volatility ha sido agregado al PATH."

Aquí hay algunas otras formas de colorear la salida en bash:

Utilizando la utilidad "tput": Este comando proporciona una variedad de funciones para controlar la terminal, incluyendo la capacidad de establecer el color de la salida. Por ejemplo, para establecer el color de la salida en rojo, puedes usar el siguiente comando:
Copy code
echo -e "$(tput setaf 1)Este es un texto en rojo$(tput sgr0)"
Utilizando el comando "echo": El comando echo también proporciona la capacidad de establecer el color de la salida mediante el uso de códigos de escape. Por ejemplo, para establecer el color de la salida en azul, puedes usar el siguiente comando:
Copy code
echo -e "\033[34mEste es un texto en azul\033[0m"
Utilizando el comando "printf": Al igual que el comando echo, printf también proporciona la capacidad de establecer el color de la salida mediante el uso de códigos de escape. Por ejemplo, para establecer el color de la salida en verde, puedes usar el siguiente comando:
Copy code
printf "\033[32mEste es un texto en verde\033[0m\n"
Espero que estos ejemplos te ayuden a entender cómo colorear la salida en bash.
